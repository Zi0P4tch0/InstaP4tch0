# InstaP4tch0

Modded Instagram for non-jailbroken iDevices.

Min iOS version: `10.0`.

Compiled using iOS `12.2` SDK.

Supported architectures: `armv7 arm64`.

Made possibile by `theos-jailed` and `CydiaSubstrate`.
